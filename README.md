# Tarik SP2DK

## Instalasi

- Instal [Node.js](https://nodejs.org/en/download)
- Buka cmd sebagai admin
- Jalankan perintah berikut:
```bash
> npm install -g git+https://github.com/cahyo6012/tarik-sp2dk.git
```

## Penggunaan

```bash
> tarik-sp2dk [-u USERNAME] [-p PASSWORD] [-o OUTPUT]
# atau
> tarik-sp2dk
```
